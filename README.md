# صيدلية القدس - Jerusalem Pharmacy

تم بناء هذا الموقع باستخدام:
- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (للحركات التفاعلية)
- **Lucide React** (للأيقونات)

## المميزات:
- تصميم احترافي (Medical Clean) باللون الأزرق الطبي (#0056b3) والتركواز.
- دعم كامل للغة العربية والاتجاه من اليمين لليسار (RTL).
- فلترة متقدمة للمنتجات تشمل 20 قسماً طبياً.
- صفحة خاصة لإرسال الوصفات الطبية مع نموذج تحميل صور.
- متوافق تماماً مع جميع الأجهزة (Mobile & Desktop).

## طريقة التشغيل:

1. قم بفتح المجلد في محرر الأكواد (VS Code).
2. قم بتثبيت المكتبات اللازمة:
   ```bash
   npm install
   ```
3. ابدأ تشغيل بيئة التطوير:
   ```bash
   npm run dev
   ```
4. افتح المتصفح على: [http://localhost:3000](http://localhost:3000)

## الهيكل التنظيمي:
- `src/app`: يحتوي على الصفحات (الرئيسية، صفحة الوصفة).
- `src/components`: المكونات التفاعلية (Header, Footer, Hero, Sidebar, Form).
- `src/constants`: بيانات الأقسام والمنتجات والماركات.
- `src/app/globals.css`: التنسيقات العالمية وإعدادات الخطوط (Tajawal).
